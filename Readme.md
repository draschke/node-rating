Node Rating
===========
Rating an Hit system

Dependencies
------------

* Node 0.2.x+
* npm
* express
* nodeunit
* mongoose (and MongoDB server)

Running
-------
$ node server.js

curl -X POST http://localhost:3000/:context/:subject/:id
